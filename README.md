# Text Fragment URL Generator

[👉 Open the generator](https://link-to-text.github.io/)

Create links that highlight and scroll to specific text on a web page using the [Text Fragment](https://web.dev/text-fragments/) syntax (`#:~:text=`).  
Supported in Chrome and other Chromium-based browsers.

## ✨ Features
- 🔗 Generate highlightable links with `#:~:text=...`
- 📚 Great for:
    - Sharing quotes or key passages
    - Linking to documentation
    - Academic citations
    - Support tickets
    - Blog posts and social media

## 🚀 How it works
Text Fragments use a special URL format that scrolls to and highlights the exact text you specify: `https://example.com/#:~:text=your%20highlighted%20text`

Our tool helps you generate this URL automatically. Just paste the base URL and text you want to highlight — and copy the result.
