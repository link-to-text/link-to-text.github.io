What is a Text Fragment link?

A Text Fragment (scroll-to-text) link uses the #:~:text= syntax to automatically scroll to and highlight specific text on any web page — no browser extension required.

Supported in Chrome, Edge, Opera, and other Chromium-based browsers. Perfect for sharing quotes, linking to documentation, adding academic citations, or pointing directly to the relevant part of a long article.

Use prefix and suffix to disambiguate when the same phrase appears multiple times on a page. The fragment format is: prefix-,start,end,-suffix.
